# 🚀 What I'm Building

### DexSpeak

An AI-powered English learning platform focused on **contextual reading and vocabulary understanding**.

Features:

- 📚 EPUB / article reading
- 🧠 AI word explanation
- 🔍 Contextual vocabulary analysis
- 🔊 TTS pronunciation
- 📈 spaced repetition learning

Tech stack:

- FastAPI backend
- MongoDB + Redis
- Cloudflare R2 storage
- OpenAI / DeepSeek / LLM APIs
- iOS native client

🌐 https://dexspeak.com
